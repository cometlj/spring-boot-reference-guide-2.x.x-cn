# 9. 系统要求

Spring Boot 2.0.0 RELEASE需要 Java 8 或者 9的版本以及Spring Frameword 5.0.4 RELEASE版本或以上。明确指定编译支持为Maven 3.2+和Gradle 4.

## 9.1 Servlet容器

Spring Boot支持一下内置的Servlet容器：

| 名称 | Servlet版本 |
| :--- | :--- |
| Tomcat 8.5 | 3.1 |
| Jetty 9.4 | 3.1 |
| Undertow 1.4 | 3.1 |

你也可以将Spring Boot应用部署在任何兼容Servlet 3.0+的容器中。
